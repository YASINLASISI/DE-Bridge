# **App Name**: DE-Bridge: Diaspora Expertise Network

## Core Features:

- User Authentication and Profiles: Secure authentication with Firebase for Seekers and Experts, detailed profile creation, credential verification for Experts.
- Expert Discovery & Booking: Browse experts by category, filter by multiple parameters, real-time availability calendar and booking system.
- Secure Payment & Escrow: Paystack integration for payments, escrow system managed via Firestore, automated payouts, refund mechanism.
- Virtual Session Management: Integrated video call functionality, real-time chat, session timer, post-session rating/review system, document upload.
- AI-Powered Medical Translator Tool: LLM-based tool that extracts and standardizes medical data from Nigerian lab results using Google's Gemini API, highlights critical values, and generates summaries for diaspora doctors.
- Smart Expert Matching: Algorithm that suggests top experts based on seeker needs, expert availability, price compatibility, ratings, and language match.
- Admin Dashboard: Admin interface for verifying expert credentials, managing bookings/payments/disputes, platform analytics, and commission rates.

## Style Guidelines:

- Primary color: Deep Blue (#1E40AF) for trust, professionalism, and healthcare focus.
- Secondary color: Emerald Green (#10B981) to represent success, growth, and connection.
- Accent color: Warm Orange (#F59E0B) evoking energy, African roots, and warmth.
- Neutral colors: Dark Gray (#0F172A) for text/headers, and Light Gray (#F8FAFC) for backgrounds.
- Headings: 'Inter' font (sans-serif), bold (700-800), tracking-tight.
- Body: 'Inter' font (sans-serif), regular to medium weight (400-500).
- Use Lucide React icons; consistent and modern style.
- Generous whitespace with spacing (py-20, py-32), max-width containers (max-w-7xl), and grid layouts.
- Subtle fade-in and slide-up animations for page transitions. Use framer-motion for all interactions.
